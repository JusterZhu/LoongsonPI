# LoongsonPI

本仓库由龙芯派爱好者建立的非官方龙芯派相关的生态仓库，一个致力于推动龙芯处理器及其生态系统发展的平台。在这里，你可以找到关于龙芯处理器的最新资讯、技术文档、开发工具、生态软件示例。（所有资料均来自龙芯开源社区）

| 名称  | 说明                 | 备注               |
| ----- | -------------------- | ------------------ |
| docs  | 龙芯派相关文档       | 使用手册、产品参数 |
| tools | 龙芯派相关文档工具。 | 烧录工具           |
| linux | 内核源码             |                    |
